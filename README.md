# Digital Clock

This project is a simple digital clock that displays the current time and dynamically changes the background image based on the time of day.


## Live Demo

Check out the live demo of the digital clock [here](). 

## Features

- Displays the current time in a 12-hour format with AM/PM indicators.
- Updates every second to ensure the time is accurate.
- Changes the background image based on the time of day:
  - **Morning (6 AM - 12 PM):** Displays a morning-themed image.
  - **Afternoon (12 PM - 6 PM):** Displays an afternoon-themed image.
  - **Evening (6 PM - 9 PM):** Displays an evening-themed image.
  - **Night (9 PM - 6 AM):** Displays a night-themed image.

## Setup

1. Clone the repository or download the files.

2. Ensure you have the following images in the same directory as your HTML file, or adjust the paths in `app.js`:
   - `morning.jpg`
   - `afternoon.jpg`
   - `evening.jpg`
   - `night.jpg`

3. Open `index.html` in your web browser to see the digital clock in action.

## Files

- `index.html`: The main HTML file containing the structure of the digital clock.
- `style.css`: Contains the styling for the digital clock.
- `script.js`: Contains the JavaScript code for updating the time and changing the background image.
- `morning.jpg`, `afternoon.jpg`, `evening.jpg`, `night.jpg`: Background images for different times of the day.
